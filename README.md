# Projeto Análise Exploratória de Dados:
O propósito deste projeto é realizar uma Análise Exploratória de Dados (AED) nos conjuntos de dados da plataforma Instacart, um serviço de entrega de supermercado fictício. O objetivo principal é limpar e processar os dados para entender os hábitos de consumo dos clientes, identificando padrões como Os itens mais vendidos, o comportamento de adição de itens ao carrinho (quais são escolhidos primeiro, por exemplo), e principalmente a higienização de dados brutos, lidando com valores ausentes e duplicados para garantir análises precisas.

# Resultados:

A maioria das compras no Instacart é composta por alimentos saudáveis, como frutas e verduras.

O item mais vendido da plataforma é a banana.

Os 20 produtos mais adicionados em primeiro lugar no carrinho são majoritariamente itens de hortifruti e produtos básicos como água e leite.

# Tecnologias utilizadas:
